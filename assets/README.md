# Assets Placeholder

Los siguientes archivos deben ser imágenes reales para que Expo funcione correctamente:

## Requeridos por Expo:
- `icon.png` - Icono de la app (1024x1024 px)
- `splash.png` - Pantalla de inicio (1242x2436 px)
- `adaptive-icon.png` - Icono adaptativo Android (1024x1024 px)
- `favicon.png` - Favicon para web (16x16 o 32x32 px)

## Solución Temporal

Puedes usar estos comandos para generar imágenes placeholder o descargar iconos de ajedrez gratis desde:
- https://www.flaticon.com/search?word=chess
- https://www.iconfinder.com/search?q=chess
- https://icons8.com/icons/set/chess

## Generación Automática

Si tienes un solo icono (icon.png de 1024x1024), Expo puede generar los demás automáticamente.

Por ahora, Expo usará iconos por defecto si estos archivos no están presentes.
